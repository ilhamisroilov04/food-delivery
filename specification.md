# Техническое задание на дипломную работу "Учет заказов на доставку еды – клиент, блюда, статус заказа"
## Данные
### Доставщик еды
* id Доставщика
* номер телефона
* имя
* паспорт
### точка выдачи
* id выдачи
* адрес
* координаты (?)
* название заведения
### Клиент
* id клиента
* адрес
* местоположение
* номер телефона
* способ оплаты

### статус заказа
* id заказа
* заказ оформлен
* заказ оплачен
* доставка в пути

* добавить клиента
