# Техническое задание на дипломную работу "Учет заказов на доставку еды – клиент, блюда, статус заказа"
## Данные
### Доставщик еды
* id Доставщика
* номер телефона
* имя
* паспорт
--------------------------
### точка выдачи
* id выдачи
* адрес
* координаты (?)
* название заведения
-----------------------
### Клиент
* id клиента
* адрес (куда доставить заказ)
* местоположение (доставки)
* номер телефона (клиента)
* способ оплаты
----------------------------
### статус заказа
* id заказа
* добавить заказ
* добавить курьера
* заказ оформлен
* заказ оплачен
* доставка в пути
* доставка доставлена
* общая сумма заказов
-------------------
