# ТЗ 
# Техническое задание на дипломную работу "Учет заказов на доставку еды – клиент, блюда, статус заказа"
## Данные
### Доставщик еды
* id Доставщика
* номер телефона
* имя
* паспорт
--------------------------
### точка выдачи
* id выдачи
* адрес
* координаты (?)
* название заведения
-----------------------
### Клиент
* id (PK)
* имя
* номер телефона (клиента)
* email
* дата регистрации
----------------------------
### Заказ
* id
* клиент id(FK)
* адрес доставки
* способ оплаты
* дата создания
* статус
* общая сумма
-------------------
## Действия
### добавить клиента
* id (PK)                                                                      
* номер телефона
* имя
* email (клиента)
* дата регистрации
* дата (последнего заказа)
* коментарий для курьера
---------------------------------
## Ресторан
### добавить заведение
* id (PK)
* название 
* адрес 
* график работы
* рейтинг
-----------------------------------
## Состав заказа
### Что туда входит
*  id (PK)
*  заказ id (FK)
*  блюдо id (FK)
*  количество
*  цена на момент заказа
-------------------------------------
